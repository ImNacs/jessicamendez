## 2. Tipografías Recomendadas

### Sans-Serif Modernas (Transmiten Profesionalismo + Modernidad)
- **Inter**: Excelente legibilidad, feeling tecnológico-accesible
- **Poppins**: Geométrica, amigable pero profesional
- **Nunito**: Redondeada, orgánica, accesible
- **DM Sans**: Moderna, limpia, versátil
- **Manrope**: Semi-geométrica, contemporánea

### Serif Elegantes (Credibilidad + Tradición)
- **Playfair Display**: Para titulares con autoridad
- **Lora**: Equilibrada, académica, confiable
- **Libre Baskerville**: Clásica, seria, profesional
- **Cormorant Garamond**: Elegante, sofisticada

### Combinaciones Recomendadas para Brochure:
```
Combinación 1 (Moderno-Profesional):
- Titulares: Playfair Display Bold
- Cuerpo: Inter Regular/Medium
- Subtítulos: Inter SemiBold

Combinación 2 (Accesible-Contemporáneo):
- Titulares: Poppins Bold
- Cuerpo: Nunito Regular
- Datos/Stats: DM Sans Medium

Combinación 3 (Serio-Corporativo):
- Titulares: Libre Baskerville
- Cuerpo: DM Sans Light
- Acentos: DM Sans SemiBold
```

### Evitar:
- Comic Sans, Papyrus (clichés negativos)
- Tipografías excesivamente decorativas
- Fuentes "eco" con hojas integradas (greenwashing visual)

## 3. Estilos de Iconografía e Ilustración

### Tendencias en Iconografía 2024-2025:

**A. Line Art Minimalista**
- Iconos de línea fina (1-2px)
- Esquinas redondeadas suaves
- Un solo color o duotono
- Estilo coherente en todo el set

**B. Iconos "Organic Geometry"**
- Formas geométricas con bordes suavizados
- Círculos imperfectos, formas orgánicas
- Sensación de hecho a mano pero refinado

**C. Ilustraciones Isométricas**
- Representación de procesos ambientales
- Infografías técnicas con estilo visual atractivo
- Buenos para explicar conceptos complejos

**D. Flat Design 2.0**
- Flat con sutiles gradientes
- Sombras suaves
- Profundidad sin ser 3D completo

### Temas de Iconografía para Consultoría Ambiental:
- Hojas estilizadas (no realistas, abstractas)
- Gotas de agua geométricas
- Círculos de economía circular
- Flechas de reciclaje rediseñadas
- Gráficos de crecimiento con elementos naturales
- Manos sosteniendo elementos (cuidado)
- Lupa + naturaleza (análisis ambiental)
- Documentos + check marks (cumplimiento)

### Ilustraciones Recomendadas:
- **Estilo "Flat Nature"**: Ilustraciones de paisajes simplificados
- **Abstract Botanical**: Formas vegetales abstractas
- **Data Visualization Art**: Gráficos como elementos decorativos
- **Pattern Design**: Patrones de elementos naturales sutiles

## 4. Tendencias en Fotografía para Sector Ambiental

### Estilos Fotográficos Trending:

**A. Fotografía Natural Auténtica**
- Luz natural, no sobreproducida
- Momentos reales, no poses
- Personas trabajando en campo
- Texturas naturales en primer plano

**B. Aéreas/Drone**
- Paisajes desde arriba
- Patrones naturales
- Escala y contexto
- Impacto visual dramático

**C. Macro/Detalle**
- Texturas de hojas, corteza, agua
- Detalles de biodiversidad
- Enfoque selectivo artístico

**D. Estilo Documental**
- Equipos trabajando en proyectos reales
- Procesos de consultoría
- Interacción con comunidades
- Behind-the-scenes auténtico

### Tratamiento de Color en Fotografía:
- **Desaturación leve**: Tonos más naturales, menos saturados
- **Filtros terrosos**: Warmth natural sin ser naranja
- **Alto contraste suave**: Blancos limpios, negros profundos pero no duros
- **Estilo "film"**: Granulado sutil, colores vintage suave

### Evitar en Fotografía:
- Stock photos genéricas de hojas/manos con tierra
- Oversaturation de verdes
- Fotos de banco de imágenes obvias
- Imágenes sin contexto o muy abstractas

## 5. Elementos Naturales vs Minimalismo

### Tendencia Actual: "Natural Minimalism" o "Organic Minimalism"

**El equilibrio perfecto incluye:**

**Elementos Naturales (usados con moderación):**
- Texturas de papel kraft o reciclado como fondo
- Patrones botánicos sutiles en márgenes
- Fotografías con mucho espacio negativo
- Ilustraciones de línea de elementos naturales
- Watermarks vegetales muy suaves

**Principios Minimalistas:**
- Mucho espacio en blanco (white space)
- Jerarquía visual clara
- Máxima legibilidad
- Grid system bien definido
- Menos es más en decoración

### Proporción Recomendada:
- 70% espacio limpio/minimalista
- 20% elementos visuales funcionales (fotos, iconos, gráficos)
- 10% elementos decorativos naturales

### Cómo Integrar:
```
Layout de página:
- Fondo: Color sólido o textura muy sutil
- Márgenes: Amplios (mínimo 15-20mm)
- Elementos naturales: Esquinas, separadores de sección
- Fotografía: Enmarcada con generoso espacio
- Iconos: Pequeños, funcionales, no decorativos
```

## 6. Cómo Transmitir Profesionalismo + Conciencia Ambiental

### Estrategias de Diseño:

**A. Credibilidad Visual**
- Uso de datos y estadísticas con visualización atractiva
- Certificaciones y sellos de calidad prominentes
- Testimoniales con fotos reales
- Timeline de proyectos completados
- Logos de clientes/partners

**B. Balance Tonal**
- 60% elementos corporativos/profesionales
- 40% elementos eco/ambientales
- Evitar ser "demasiado verde"

**C. Estructura Profesional**
- Índice claro
- Numeración de páginas
- Headers consistentes
- Pie de página con contacto
- Llamadas a la acción claras

**D. Lenguaje Visual de Consultoría**
- Diagramas de proceso
- Matrices de servicios
- Flujos de trabajo
- Casos de estudio estructurados
- ROI y métricas visualizadas

### Elementos que Comunican Ambos Valores:

| Profesionalismo | Conciencia Ambiental |
|-----------------|----------------------|
| Tipografía serif clásica | Paleta de colores terrosos |
| Layout estructurado | Texturas orgánicas sutiles |
| Gráficos de datos | Iconografía natural |
| Certificaciones | Mensajes de impacto |
| Casos de estudio | Fotografía auténtica |
| Llamadas a acción claras | Papel sostenible (físico) |

## 7. Ejemplos de Branding Exitoso en Consultoría Ambiental

### Características de Marcas Exitosas:

**Patagonia (Referencia de Autenticidad)**
- Paleta: Azules profundos, naranjas terrosos
- Tipografía: Sans-serif limpia, directa
- Fotografía: Documental, auténtica, dramática
- Mensaje: Activismo real, no solo marketing

**Interface (Diseño Corporativo Sostenible)**
- Paleta: Verdes sofisticados, grises
- Enfoque en datos y métricas de sostenibilidad
- Visualización de impacto ambiental
- Balance corporativo-eco

**Ecologic Brands**
- Minimalismo extremo
- Colores neutros con acentos verdes
- Énfasis en materiales
- Transparencia visual

### Estructura de Brochure Exitoso:
```
1. Portada impactante (fotografía + logo)
2. Mensaje de misión (1 página)
3. Servicios (iconos + descripciones cortas)
4. Metodología (diagrama visual)
5. Casos de éxito (fotos + datos)
6. Equipo (fotos profesionales)
7. Certificaciones/Partners
8. Contacto + CTA
```

## 8. Qué Evitar: Greenwashing Visual y Clichés

### Clichés Visuales a Evitar:

**Iconografía Sobreusada:**
- Planeta tierra con manos abrazándolo
- Hojas en forma de corazón
- Bombilla con planta dentro
- Reciclaje símbolo como único elemento
- Huellas ecológicas literales
- Árboles genérico de clip art

**Colores Cliché:**
- Verde neón/lima brillante (#00FF00)
- Verde "tóxico" saturado
- Combinación verde + marrón sin sofisticación
- Degradados arcoíris "eco"

**Fotografía Cliché:**
- Manos sosteniendo plántula
- Persona abrazando árbol
- Gotas de agua perfectas en hojas
- Oso polar en hielo derretido (sin contexto)
- Grupos sonriendo plantando árboles (stock)

**Elementos de Greenwashing Visual:**
- Exceso de verde sin sustancia
- Sellos "eco" autodeclarados sin certificación
- Números vagos ("ayudamos al planeta")
- Imágenes de naturaleza sin conexión con servicio
- Promesas visuales sin respaldo

### Señales de Alerta en Diseño:
1. Todo es verde
2. No hay datos concretos
3. Imágenes genéricas sin contexto
4. Falta de transparencia visual
5. Exceso de símbolos eco sin explicación
6. Inconsistencia entre mensaje y acción visual

### Cómo Evitar Greenwashing:
- Mostrar resultados específicos con números
- Usar fotos de proyectos reales
- Incluir certificaciones verificables
- Ser específico sobre metodologías
- Mostrar el equipo real
- Documentar procesos, no solo resultados

## 9. Recomendaciones Específicas para Brochure de Consultoría Ambiental

### Paleta Recomendada:
```
Primario: #2D5016 (Verde bosque profundo)
Secundario: #F7F4EF (Crema muy claro)
Acento 1: #006994 (Azul profesional)
Acento 2: #B66A50 (Terracotta para destacados)
Texto: #2C2C2C (Carbón suave)
```

### Tipografía Sugerida:
```
Titulares: Playfair Display Bold (autoridad)
Subtítulos: Inter SemiBold (claridad)
Cuerpo: Inter Regular (legibilidad)
Datos/Estadísticas: DM Sans Medium (precisión)
```

### Estructura Visual:
1. **Portada**: Fotografía dramática de paisaje + logo minimalista
2. **Página de misión**: Mucho espacio blanco, una frase potente
3. **Servicios**: Grid con iconos line-art + descripción concisa
4. **Proceso/Metodología**: Diagrama de flujo elegante
5. **Casos de estudio**: Antes/después con datos reales
6. **Equipo**: Fotos profesionales pero naturales
7. **Impacto**: Infografía con métricas concretas
8. **Contacto**: CTA claro, redes, certificaciones

### Elementos Diferenciadores:
- Incluir QR a reportes de impacto
- Mostrar timeline de proyectos
- Destacar certificaciones con diseño elegante
- Usar testimoniales con foto y cargo
- Incluir "números que importan" (árboles salvados, CO2 reducido, etc.)
