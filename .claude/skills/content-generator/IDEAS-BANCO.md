# Banco de Ideas de Contenido

## Cómo Usar Este Banco

1. **Elige una categoría** según tu enfoque actual
2. **Selecciona el objetivo** (Atraer, Involucrar, Deleitar)
3. **Adapta el título** a tu voz y audiencia
4. **Marca como usada** cuando publiques
